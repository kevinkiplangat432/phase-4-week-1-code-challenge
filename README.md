# Superheroes API

A Flask REST API for tracking superheroes and their superpowers.

## Features

- Manage superheroes with their secret and public identities
- Track various superpowers with detailed descriptions
- Associate heroes with multiple powers and specify strength levels
- Full CRUD operations for all resources
- Data validation and error handling
